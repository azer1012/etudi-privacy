Politique de confidentialité – ETUDi

Dernière mise à jour : 2026

1. Données collectées

Nous collectons les informations suivantes :
- Nom et informations de profil
- Numéro de téléphone (si fourni)
- Données liées aux élèves, séances et paiements
- Données d’utilisation de l’application

2. Utilisation des données

Les données sont utilisées uniquement pour :
- Fournir et améliorer les fonctionnalités de l’application
- Gérer les comptes utilisateurs
- Assurer le bon fonctionnement des services

3. Partage des données

Nous ne vendons ni ne partageons vos données personnelles avec des tiers.

4. Sécurité

Les données sont sécurisées et chiffrées lors de leur transmission.

5. Conservation des données

Les données sont conservées tant que le compte est actif.

6. Vos droits

Vous pouvez demander la suppression de vos données à tout moment en nous contactant.
